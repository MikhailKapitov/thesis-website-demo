# Demo website

A demo for our thesis project.

## Setup (Local)

1. Install the dependencies.
   ```bash
   npm install
   ```
2. Pull and update the submodule.
   ```bash
   cd ./public/map
   git pull origin main
   cd ../..
   ```
3. Run the website.
   ```bash
   npm run dev
   ```
