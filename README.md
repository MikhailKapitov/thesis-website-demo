# Demo website

A demo for our thesis project.

## Setup (Local)

1. Install the dependencies.
   ```bash
   npm install
   ```
2. Init the submodule. Only needs to be done once.
   ```bash
   git submodule update --init --recursive
   ```
3. Run the website.
   ```bash
   npm run dev
   ```
